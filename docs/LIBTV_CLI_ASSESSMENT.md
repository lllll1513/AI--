# LibTV CLI 对 AI短剧工作台的帮助评估

安装状态：
- `libtv-cli` skill 已存在，版本与用户提供的 zip 一致。
- `libtv.exe` 已安装，可正常运行。
- 当前 CLI 版本：1.1.1。
- 当前账号状态：未登录。

## 它能帮我们做什么

LibTV CLI 适合作为“外部画布自动化工具”，尤其适合下面这些事：

- 自动创建 LibTV 工作区和画布。
- 自动上传角色图、场景图、参考图、音频、视频等素材。
- 自动创建画布节点。
- 自动创建分组，比如“人物素材”“场景素材”“第01镜”“第02镜”。
- 自动把节点按工作流连成画布结构。
- 自动读取画布节点列表和资源状态。
- 后续可把我们工作台里的镜头、提示词、素材包同步到 LibTV 画布。

## 对黑神话吕布这类项目的价值

如果要用 LibTV，它最适合做：

- 每个镜头一个分组。
- 每个分组里放该镜头用到的角色参考、场景参考、动作提示词、视频节点。
- 把“吕布、三名强敌、杂兵、武器、战场、特效”按资产类型批量上传。
- 把工作台生成的分镜脚本同步成 LibTV 画布节点，方便在可视化画布里继续跑。

这对“素材很多、镜头很多、需要严格绑定参考图”的项目有明显帮助。

## 需要注意的限制

- 使用 LibTV 的远端能力需要登录。
- 真正生成图片、视频或运行某些节点，大概率会消耗平台额度或产生费用。
- 它不应该替代我们自己的 AI短剧工作台地基。
- 默认不应该自动触发付费生成，除非用户明确确认。

## 推荐定位

主工作台：AIComicBuilder / AI短剧工作台

负责：
- 项目库
- 角色库
- 分镜
- 提示词
- 镜头视频
- 自动质检
- 自动剪辑
- 本地合成

LibTV：

作为可选外部执行器，用于：
- 把项目同步到 LibTV 画布。
- 批量上传素材。
- 批量创建画布节点。
- 需要时借用 LibTV 的画布工作流。

## 后续集成方案

第一步：只读和准备
- 检查 `libtv --version`。
- 检查登录状态。
- 不自动登录，不自动生成。

第二步：导出到 LibTV
- 从 AI短剧工作台导出素材包。
- 生成 LibTV CLI 命令清单。
- 用户确认后再执行上传和建节点。

第三步：画布同步
- 一个项目对应一个 LibTV 工作区或画布。
- 一个镜头对应一个 LibTV 分组。
- 镜头内节点包含：角色参考、场景参考、提示词、视频生成节点、输出视频。

第四步：费用保护
- 所有可能消耗额度的命令都必须先列出执行计划。
- 用户确认后再运行。
- 每次运行后记录输出和失败原因。

## 当前结论

它有帮助，但不是核心地基。最适合后续作为“LibTV 画布桥接器”，让 Codex 可以把我们整理好的素材、分镜和提示词自动铺到 LibTV 画布上。
